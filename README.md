# Stress Test Skill

A structured failure-first review skill for plans, launches, strategies, delivery commitments, and organizational decisions.

This skill helps teams assume a plan has failed six months from now, then work backward to identify likely failure modes, hidden assumptions, base rates, early warning signs, and concrete revisions.

## Usage

Ask Claude or a compatible AI assistant:

- "Stress test this release plan"
- "Stress test this AI rollout"
- "What could kill this plan?"
- "Find the blind spots in this strategy"
- "Poke holes in this proposal"
